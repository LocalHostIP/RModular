# Instalar darknet

git clone https://github.com/AlexeyAB/darknet

cd darknet

mkdir build_release

cd build_release

cmake ..
cmake --build . --target install --parallel 8

## reinstalar cmake
sudo apt remove cmake

Visit https://cmake.org/download/ and download the latest bash script.
